# Общее описание

- Назначение: платформа для записи к мастерам, с Telegram WebApp и веб‑интерфейсом.
- Роли: мастер, клиент, суперадмин (админка — Filament).
- Каналы: единый бот с двумя WebApp‑кнопками («Мастер», «Клиент»). Веб‑версии доступны по URL.
- Технологии: Laravel 12 (PHP 8.3+), Inertia.js + Vue 3, Filament 3, PostgreSQL, Redis (кеш/очереди/сессии), Docker.
- Архитектура: тонкие контроллеры → Actions/Services → JsonResource; валидация через FormRequest.
- Основные страницы: «Календарь мастера», «Настройки мастера», «Бронирование клиента».
- Уведомления: напоминания в Telegram для мастеров и клиентов через очереди.

Как это работает кратко:
- Клиент открывает клиентский WebApp, выбирает город/услугу/мастера и записывается; его данные сохраняются и сегментируются.
- Мастер входит через мастерский WebApp. Регистрация мастера — отдельный онбординг с обязательными настройками; после завершения профиль активируется.
